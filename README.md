# Telecom_Customer_Churn_Analysis
Built a customer churn analysis project using SQL and Power BI. Performed data cleaning, feature engineering, and created interactive dashboards to identify churn drivers, analyze customer behavior, and deliver actionable insights to improve retention

# Database & Tools used
- Postgre SQL (pgAdmin4)
- Power BI

# Business Problem
## Customer churn significantly impacts revenue and long-term growth

### The objective of this analysis is to:
- Identify key factors driving customer churn
- Analyze customer behavior patterns
- Provide actionable recommendations to improve retention
# Dataset Description 
-<a href="https://github.com/Tusharnjaiswal/Telecom_Customer_Churn_Analysis/blob/main/telecom_customer_churn.csv">dataset link</a>
- Total Customers: 5.52k
- Features: Demographics, Services, Billing, Contract Details, etc
- Target Variable: Customer Status (Churned / Stayed)
#### Data was preprocessed and transformed using SQL for analysis
# Approach & Methodology
- Data Cleaning and Preparation using SQL
#### Feature Engineering:
- Churn Flag (0 = Stayed, 1 = Churned)
- Tenure Groups
- Service Count (Customer Engagement Indicator)
- Monthly Charge Segmentation
- Data Visualization using Power BI
